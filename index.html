I Try To Become A Hacker.

I consider it the best profession there is. Whether you are a pentester, bug hunter, malware analyst or security architect the core values of hacking are, as the youngsters say, pretty based. Above all it preaches understanding of concepts both on theoretical and practical levels, something extremely necessary if one wants to be a creator rather than another cog. However hacking doesn't create, it breaks. It exposes the logical fallacies of technological concepts and their executions that as a result are harmful to different groups, such as companies or the public.

Yoy might say that I'm not in a position to spout about hacker ideas when I'm still a toddler in the field. However even the opinion of a brainlet can be insightful. Just Take The Linux Pill.

I consider Linux the definitive experience for using computers, there is no excuse for not using Linux. Whether it be programming, customization, servers, efficiency, feeling of superiority, or technological freedom Linux does a much better job than MacOS or Windows. And for all you gamer boys and gals I've heard that gaming on Linux is improving exponentially, so why not try it out?

I currently run Manjaro with i3 on my desktop and Arch with dwm on my Thinkpad X230. Maybe I'll add my config files on github and add the programs I use.
